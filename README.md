# الكاشير - نظام إدارة المطابع

تطبيق ويب متقدم (PWA) لإدارة عمليات المطابع.

## المميزات
- ✅ إدارة فواتير المطابع مع تسلسل تلقائي
- ✅ إدارة الزبائن
- ✅ إحصائيات متقدمة (سنة > شهر > أيام)
- ✅ محفظة وسجل حركات
- ✅ وضع داكن
- ✅ تصدير/استيراد JSON
- ✅ يعمل بدون إنترنت (PWA)
- ✅ يعمل على الهاتف والحاسوب

## الرفع على GitHub Pages
1. أنشئ Repository جديد
2. ارفع جميع الملفات إلى الـ Root
3. اذهب إلى Settings > Pages > اختر main branch
4. سيعمل التطبيق على: `https://username.github.io/repo-name`
